# Screenshots

Drop PNGs here to make them appear in the main README. Expected filenames:

| File | What to capture |
|------|-----------------|
| `landing.png`   | the homepage (hero + stats section) |
| `dashboard.png` | the Overview page while signed in |
| `trace.png`     | a run's expanded trace tree |
| `approvals.png` | the Approvals queue with a held action |

Take them at ~1280px wide for a crisp render.
